# DExcelAssist v99

DExcelAssist は、RelaxTools 由来の必要機能と DExcelAssist 追加機能だけを 1 つの `DExcelAssist` リボンにまとめた Excel アドインです。

## v99 の方針

- 自動アップデート機能は含めません。
- Excel起動時、Windowsタスク、バックグラウンドでの更新確認は行いません。
- インストール時に削除・置換する対象は `DExcelAssist.xlam` のみです。
- RelaxTools など他の Excel アドインは削除しません。
- バッチは `DExcelAssist.bat` ひとつだけです。

## メニュー

1. インストール/修復
2. 診断
3. アンインストール
4. Excel残留プロセスを強制終了
5. リリース用ファイル作成
0. 終了


## v99 の変更点

- `シート一覧作成` の表示名を `シート一覧` に変更しました。
- `シート一覧` シートから `種別` 列を削除しました。
- 指定フォルダ内のファイルを一覧化する `ファイル一覧` 機能を追加しました。
  - フォルダ選択ダイアログで対象フォルダを選択
  - サブフォルダを含めるか選択
  - `ファイル一覧` シートに、ファイル名・拡張子・フォルダ・フルパス・サイズ・更新日時・作成日時を出力
  - ファイル名は実ファイルへのリンクに設定

## インストール手順

1. Excelファイルをすべて保存して閉じます。
2. `DExcelAssist.bat` を実行します。
3. `1: インストール/修復` を選択します。
4. Excelを起動し、`DExcelAssist` リボンを確認します。

## 注意

インストール/修復では、最初に Excel プロセスを強制終了します。開いているブックは必ず保存してから実行してください。

VBAエクスポート機能を使用する場合は、Excel側で「VBAプロジェクト オブジェクト モデルへのアクセスを信頼する」を有効にしてください。


## 既存更新内容

- リリース用ファイル作成時に、バッチ実行者がバージョンを入力できるようにしました。
- 入力したバージョンは `VERSION.txt`、配布ZIP名、`main_branch_upload` に反映されます。


## 既存追加機能

- シート一覧作成：各シートの一覧を作成し、シート名を各シートへのリンクにします。
- Backlogタブ：親課題でグループ化を追加しました。
- VBAエクスポート：フォルダ選択ダイアログで出力先を選びやすくしました。


## 既存更新内容

- フォルダツリーの出力先選択UIを改善しました。
- リボンの「フォルダツリー」は、フォルダ選択ダイアログで作成先フォルダを選択できます。
- 選択範囲の文字列をもとに、選択した出力先フォルダ配下へフォルダ階層を作成します。


## 既存更新内容

- フォルダツリーの動作を修正しました。
- セル文字列からフォルダを作成する動作ではなく、選択したフォルダ配下のフォルダ/ファイル構造をExcelシートへ挿入する動作に変更しました。
- フォルダ選択ダイアログで取得元フォルダを選択できます。
- ファイルを含めるかどうかを選択できます。
- 出力結果はアクティブセル起点に、ツリー文字列・種類・パス・更新日時・サイズ(KB)として出力します。
- ツリー文字列にはフォルダ/ファイルへのリンクを設定します。


## v99 の変更点

- フォルダツリーの表示名を「フォルダツリー」に変更しました。
- フォルダツリーの出力先をアクティブセル起点ではなく、新規作成/再作成する「フォルダツリー」シートに変更しました。
