# DelaxTools main branch upload files

このフォルダは DelaxTools.bat の「5: リリース用ファイル作成」で自動作成されます。
GitHub main ブランチ直下には、この main_branch_upload フォルダの中身だけを配置してください。

配置するファイル:
- VERSION.txt
- version.json
- DelaxToolsInstaller.zip
- README.md

DelaxTools の「アップデート確認」は VERSION.txt を確認し、新しいバージョンがある場合に DelaxToolsInstaller.zip をダウンロードしてインストーラを起動します。
