# DExcelAssist v89 統合リボン版

## 変更内容

- すべての機能を `DExcelAssist` という1つのExcelリボンタブに統合しました。
- これまで分かれていた `RelaxTools` / `RelaxShapes` / `RelaxApps` タブは表示しません。
- 既存の必要機能と追加機能は、`DExcelAssist` タブ内のグループとして配置しています。

## 含まれる主な機能

- お気に入り
- 全シートA1保存
- 検索/置換/修飾
- ショートカット割当
- 付箋関連
- フォルダツリー作成
- VBAエクスポート
- 休日シート作成
- 全シート倍率100%
- 半角⇒全角変換
- 全角⇒半角変換
- 列幅自動調整

## インストール手順

1. Excelをすべて閉じる
2. `DExcelAssist.bat` を実行
3. `4: Excel残留プロセスを強制終了` を実行
4. `1: インストール/修復` を実行
5. Excelを起動し、`DExcelAssist` タブを確認

## 注意

VBAエクスポートおよびインストール時のVBAモジュール取込には、Excelの「VBAプロジェクト オブジェクト モデルへのアクセスを信頼する」が必要です。


## v89 変更点

- `1: インストール/修復` の最初に Excel プロセスを強制終了する処理を追加しました。
- 手動で `4: Excel残留プロセスを強制終了` を実行しなくても、インストール時に自動で Excel を終了します。


## v89 変更点

- `1: インストール/修復` 実行時、同名アドイン（DExcelAssist/RelaxTools）がExcelに登録済みなら先に登録解除してから追加します。
- 同名アドインが存在しない場合は、削除処理を行わず通常どおり追加のみ行います。
- AddInsフォルダ、XLSTART、OPEN起動登録、Add-in Managerの同名登録を整理してから再登録します。


## v89 追加
- 行高さ自動調整：実行シートの使用範囲の行高さを自動調整します。


## v89 修正

- インストール時に削除・解除する対象を DExcelAssist.xlam のみに限定しました。
- RelaxTools.xlam を含む他のExcelアドイン、OPEN登録、Add-in Manager登録、XLSTART配置は削除しません。
- 既存のDExcelAssistがある場合のみ解除・削除してから再追加し、なければ追加のみ実行します。
