## 実行方法

1. Excelをすべて閉じます。
2. `DelaxTools.bat` を実行します。
3. `1: インストール / 修復` を選択します。

## バッチメニュー

```text
1: インストール / 修復
2: 診断
3: アンインストール
4: 配布ファイル作成
0: 終了
```

## 注意

Excelの強制終了メニューは削除しています。インストール前にExcelを閉じる必要がある場合は、手動で閉じてください。
